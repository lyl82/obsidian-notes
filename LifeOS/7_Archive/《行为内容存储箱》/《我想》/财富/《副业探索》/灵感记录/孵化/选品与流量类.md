 顾客如何找到你、为什么要买你

电商的第一件事其实不是卖，而是被看见。

而流量 ≠ 钱，**低质量流量只会浪费你精力和成本**。

### 🔑 关键概念：

| 概念          | 解释              | 处理问题的思路          |
| ----------- | --------------- | ---------------- |
| **蓝海品/红海品** | 红海：竞争激烈，蓝海：尚未爆发 | 适合新手的是蓝海长尾品      |
| **转化率**     | 浏览后产生购买的比例      | 图差、文案差、价格贵都会影响   |
| **点击率**     | 有多少人看到后愿意点      | 主图设计决定成败         |
| **关键词权重**   | 商品名称中关键词决定搜索曝光  | 标题是 SEO，必须研究竞争词  |
| **UV价值**    | 每一个访客能带来多少收益    | 低价引流品 + 利润爆品组合策略 |
| **加购/收藏率**  | 反映用户意愿          | 是爆款潜力的前兆指标       |
