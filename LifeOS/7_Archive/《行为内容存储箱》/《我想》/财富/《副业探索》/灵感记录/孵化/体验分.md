| 模块       | 指标         | 解释                   |
| -------- | ---------- | -------------------- |
| **服务体验** | **动态评分**   | 发货速度、描述相符、服务态度三项1-5星 |
| **店铺权重** | **动销率**    | 店铺多少商品最近30天有销量       |
|          | **退款率**    | 退款多则降权               |
|          | **纠纷率**    | 投诉、仲裁、虚假发货等越多越危险     |
| **发货指标** | **及时发货率**  | 按平台规定时间发货比例          |
|          | **物流时效**   | 平均送达时长               |
| **产品体验** | **转化率**    | 有多少人浏览后下单            |
|          | **点击率**    | 商品图吸引人程度             |
|          | **收藏/加购率** | 表示用户对商品感兴趣程度         |
