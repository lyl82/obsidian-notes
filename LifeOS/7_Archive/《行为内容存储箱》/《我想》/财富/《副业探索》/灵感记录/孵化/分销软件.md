| 功能模块     | 作用                   |
| -------- | -------------------- |
| **采集**   | 一键把1688或同行的商品信息抓取下来  |
| **上架同步** | 批量上架到你淘宝/拼多多店铺       |
| **改价规则** | 自动设置加价比例、利润空间        |
| **库存监控** | 上游商品下架/缺货，自动同步到你店里   |
| **订单同步** | 客户下单后自动同步订单信息到1688下单 |
| **发货监控** | 发货状态同步到平台，自动标发货      |
| **客服插件** | 自动回复、AI客服、多店统一接待     |
| **风控规则** | 检测违规词、低评分商品，避免限流封店   |
