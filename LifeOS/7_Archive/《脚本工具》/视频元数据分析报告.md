## 🔧 ExifTool 使用命令参考

### 1. 查看单个视频元数据
```bash
F:\downloadforsetup\exiftool-13.45_64\exiftool.exe -G1 "视频路径.mp4"
```

### 2. 提取特定字段
```bash
# 提取时间和GPS
F:\downloadforsetup\exiftool-13.45_64\exiftool.exe -CreateDate -GPSLatitude -GPSLongitude "视频路径.mp4"
```

### 3. JSON格式输出
```bash
F:\downloadforsetup\exiftool-13.45_64\exiftool.exe -json -G "视频路径.mp4"
```

### 4. 批量导出到CSV
```bash
F:\downloadforsetup\exiftool-13.45_64\exiftool.exe -csv -r -ext mp4 -ext mov "目录路径" > output.csv
```

### 5. 只显示包含GPS的文件
```bash
F:\downloadforsetup\exiftool-13.45_64\exiftool.exe -if "$GPSLatitude" -FileName -GPSPosition -r "目录路径"
```
