# Relationships Control: 社交成本与连接冗余清理

> 使用前先遵循 `1_ControlPlane/🚦_Control_IF-THEN.md` 的总闸门规则，本文件仅在全局元控制未否决时作为 Relationships 领域的局部决策参考。

| 字段 | 内容描述 |
| :--- | :--- |
| **触发条件** | 收到非核心名单消息 OR 社交账户未读数积压 OR 周末社交维护日 |
| **适用范围** | 非生产性的信息交互、人情往来、即时通讯响应 |
| **判断规则** | 1. **价值审判**：发送者不在核心名单且信息无实质价值，直接**丢弃**。<br>2. **专注保护**：处于深度工作模式，所有消息**延迟**至 21:00 集中处理。<br>3. **断舍离**：对于超过 30 天未维护且无正向反馈的关系，**执行**单向切断或最低限度回复。<br>4. **默认处理**：不确定价值的外部请求，先**延迟**处理。 |
| **输出** | **执行 / 延迟 / 丢弃** |
