# Work Control: 投入产出比(ROI)绝对审判

> 使用前先遵循 `1_ControlPlane/🚦_Control_IF-THEN.md` 的总闸门规则，本文件仅在全局元控制未否决时作为 Work 领域的局部决策参考。

| 字段 | 内容描述 |
| :--- | :--- |
| **触发条件** | 任务队列新增项 OR 收到协作请求 OR 当前任务耗时超支 20% |
| **适用范围** | 职场任务、项目开发、技术探索及会议申请 |
| **判断规则** | 1. **效率审判**：预估产出比 < 1.0 且非强制性行政任务，直接**丢弃**。<br>2. **阻塞检查**：存在外部依赖且当前无法解决，任务强制**延迟**至阻塞解除。<br>3. **优先级**：属于 S 级核心目标且资源就绪，立即**执行**。<br>4. **默认处理**：其余 B/C 级琐事，一律**延迟**至每日低能耗时段集中处理。 |
| **输出** | **执行 / 延迟 / 丢弃** |
