# Health Control: 能量代谢与物理强制执行

> 使用前先遵循 `1_ControlPlane/🚦_Control_IF-THEN.md` 的总闸门规则，本文件仅在全局元控制未否决时作为 Health 领域的局部决策参考。

| 字段 | 内容描述 |
| :--- | :--- |
| **触发条件** | 每日 07:00 唤醒检测 OR 任意时刻检测到心率变异性(HRV)异常 |
| **适用范围** | 物理身体的所有主动消耗行为（HIIT、力量训练、深度脑力工作） |
| **判断规则** | 1. **睡眠红线**：昨晚睡眠不足 6 小时，则**丢弃**今日所有高强度训练。<br>2. **生理预警**：检测到 HRV 异常或极度疲劳，则所有消耗项**延迟**至下一检测时段。<br>3. **执行标准**：状态评分 > 70% 且处于计划日，则**执行**既定项目。<br>4. **默认处理**：其余模糊情况一律**丢弃**。 |
| **输出** | **执行 / 延迟 / 丢弃** |

## 睡眠管理控制 (Sleep Management Control)

| 场景 | IF (触发条件) | THEN (执行策略) |
| :--- | :--- | :--- |
| **晨起激活** | 07:00 - 09:00 起床后 | **执行**：强光 + 冷水 + 简单运动 + 高蛋白早餐 (见 Service 1.1) |
| **白天低能** | 感到「衰弱」或本体感觉模糊 | **执行**：检查局部紧绷 + 身体扫描 + 户外光照 (见 Service 2.2) |
| **睡前准备** | 20:30 (电脑自动关机) | **执行**：进入筛选模式，过滤屏幕/逻辑/情绪刺激 (见 Service 3.2) |
| **环境切换** | 21:30 | **执行**：佩戴琥珀色镜片 + 切换暖色低照度光源 |
| **入睡失败** | 平躺 > 40 分钟无睡意 | **执行**：起床，离开卧室，进行低刺激活动 (如读古诗) |

## 行为筛选决策树 (Evening Decision Tree)

1. **第一轮 (Physiological)**: 是否存在蓝光/剧烈运动/强刺激内容？
    - **Yes** -> **丢弃 (Drop)**
    - **No** -> 进入下一轮
2. **第二轮 (Cognitive)**: 是否需要强逻辑思考/易引发情绪波动？
    - **Yes** -> **丢弃 (Drop)**
    - **No** -> **允许 (Allow)**: 手写/轻拉伸/冥想/读散文
